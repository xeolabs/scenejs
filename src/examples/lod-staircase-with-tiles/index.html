This example requires a browser that supports the HTML5 <canvas> feature.

Level-of-Detail Culling with BoundingBox Nodes

This scene contains a staircase model that switches representations as a function of the projected size of it's extents. The switching is done by a SceneJS.BoundingBox, which selects an appropriate child node for its current projected canvas size.

Drag the mouse up and down to move forwards and backwards, left and right to turn.

Move backwards and watch the staircase simplify.
