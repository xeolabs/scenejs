Using a Texture Atlas

With this technique we define one large texture containing an "atlas" of sub-textures, which we use individually on many geometries, each of which have UV coordinates that map to a different region of the texture. In a scene where there are many small textures, this has the benefit of reducing state changes on the graphics hardware by binding once, instead of for each individual texture.
