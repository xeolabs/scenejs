SceneJS > Examples > Geometry Deformation - Wobbly Teapot

This example requires a browser that supports the HTML5 <canvas> feature.

Deforming geometry vertices with the deform node

In this example we define six deformation points around a teapot, then animate the weights of those points in a funky sinusoidal ripple to make the teapot wobble.
