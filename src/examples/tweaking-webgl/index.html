Tweaking WebGL State with a Renderer Node

This demo shows off the SceneJS.Renderer node, which allows you to specifically tweak all sorts of WebGL states, such as the scissor box, blending modes and so on.


Here we have enabled backface culling and defined the vertex winding to be counter-clockwise.
