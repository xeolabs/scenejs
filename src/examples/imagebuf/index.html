Introducing the imageBuf Node

An imageBuf node renders its subgraph to a hidden image buffer, which may then be referenced as a texture source by texture nodes. An imageBuf can do this continuously, refreshing the buffer with each frame, so that the buffer works as a live a sort of video feed of the subgraph.

In this example, weÂ´re defining an imageBuf that captures a rotating teapot as a texture, which is then applied to a cube. The imageBuf recaptures the teapot with each frame, to provide an animated texture.
